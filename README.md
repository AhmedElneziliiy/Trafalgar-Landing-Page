# Trafalgar — Healthcare Landing Page

A responsive **HTML/CSS/JS landing page** for a healthcare and telemedicine service — featuring a clean medical-themed design with sections for services, app promotion, and provider listings.

## What it does

A static frontend landing page for a fictional healthcare platform called Trafalgar. Showcases the platform's features: online consultations, pharmacy, disease tracking, emergency care, and healthcare provider listings.

## Tech Stack

- **HTML5**
- **CSS3** — custom styling with a medical/health aesthetic
- **Vanilla JavaScript** — interactive elements

## Sections

- Hero / landing banner
- Key services (consultations, pharmacy, emergency, tracking)
- Healthcare provider listings
- Mobile app download promotion

## Getting Started

Open `index.html` directly in a browser — no build step required.
